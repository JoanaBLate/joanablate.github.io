This is a simple tool for creation of a font sheet.
The font sheet must have *FOUR* rows (it is not for smartphones).
You change the font family by editing the CSS code of this page.
Take a screenshot of this page. Using some image editor, extract the font sheet (WITHOUT the dashed border) from the screenshot and save it as a PNG file.
Then load the font sheet in the tool Test Font Sheet to test it.

DON'T zoom this page.
